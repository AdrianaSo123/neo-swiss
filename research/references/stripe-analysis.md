# Neo-Swiss Reference Analysis

## Reference Website: Stripe.com

**Date Analyzed:** November 7, 2025

### Neo-Swiss Principles Observed:

#### ✅ Grid System
- Clean, consistent grid-based layout
- 12-column grid system visible in desktop layout
- Consistent gutters and spacing
- Content aligns to clear grid structure
- Responsive breakpoints (desktop/tablet/mobile)

#### ✅ Typography
- Modern sans-serif typeface (consistent with Neo-Swiss)
- Clear typographic hierarchy:
  - Large, bold headlines for sections
  - Smaller, readable body text
  - Consistent font weights and sizes
- Fluid, responsive typography that scales
- High contrast (dark text on light backgrounds)

#### ✅ Color Palette
- **Minimal color scheme:**
  - Black/dark gray text (#000, #424770)
  - White/light gray backgrounds (#FFFFFF, #F6F9FC)
  - Single accent color: Stripe blue (#635BFF)
- Excellent contrast ratios
- Strategic use of color to highlight CTAs and important elements

#### ✅ Layout & Spacing
- **Generous white space** throughout the design
- Clear visual hierarchy with consistent spacing
- Modular sections with breathing room
- Content doesn't feel cramped or cluttered
- Strategic use of negative space to guide attention

#### ✅ Responsive Design
- Mobile-first approach
- Content adapts fluidly across screen sizes
- Grid collapses appropriately on smaller screens
- Maintains hierarchy and readability on all devices

#### ✅ Visual Elements
- **Minimal decoration** - function over ornament
- Clean, simple buttons and UI elements
- Subtle use of shadows and borders
- High-quality imagery used strategically
- No unnecessary visual clutter

### Key Takeaways for Our Neo-Swiss Website:
1. **Use a 12-column grid** that collapses responsively
2. **Implement generous white space** - don't crowd content
3. **Stick to 2-3 colors max** - black, white, and one accent
4. **Create clear typography hierarchy** with modern sans-serif
5. **Design mobile-first** with fluid responsive breakpoints
6. **Keep it minimal** - every element should have purpose

### Design Patterns to Emulate:
- Hero sections with large typography and clear CTAs
- Multi-column content sections
- Card-based layouts with consistent spacing
- Clean navigation with minimal links
- Footer with organized link sections
